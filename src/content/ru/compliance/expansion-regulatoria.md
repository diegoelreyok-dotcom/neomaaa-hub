## 1. ОБЩЕЕ ВИДЕНИЕ РАСШИРЕНИЯ

### 1.1 Стратегия

NEOMAAA запускается с лицензией Anjouan как быстрой точкой входа на рынок. Лицензия Anjouan позволяет работать глобально (за исключением ограниченных юрисдикций), пока формируется послужной список, клиентская база и капитал для получения лицензий в более признанных юрисдикциях.

Предлагаемый маршрут расширения:

```
ТЕКУЩАЯ ФАЗА          ФАЗА 2                ФАЗА 3              ФАЗА 4
Anjouan (AOFA)    -->  Seychelles (FSA)  --> Mauritius (FSC)  --> CySEC / FCA
                       или Mauritius (FSC)   или CySEC            (долгосрочно)
                       
Стоимость: Оплачено   Стоимость: $25-50K    Стоимость: $50-100K  Стоимость: $300K+
Время: Операционно     Время: 3-6 месяцев    Время: 6-12 месяцев  Время: 12-24 месяца
Compliance: Базовый    Compliance: Средний    Compliance: Высокий  Compliance: Очень высокий
```

### 1.2 Зачем расширяться

| Причина | Детали |
|---|---|
| Достоверность | Искушённые клиенты предпочитают брокеров с несколькими лицензиями |
| Доступ к рынкам | Определённые юрисдикции (ЕС, Великобритания, Австралия) принимают только брокеров с местной лицензией |
| Банковские отношения | Лучшие возможности banking и PSP при наличии признанных лицензий |
| Конкуренция | Конкуренты, такие как Exness и IC Markets, имеют 5+ лицензий |
| Стоимость компании | Мультирегулируемый брокер стоит значительно больше при продаже или привлечении инвестиций |

## 2. ТЕКУЩЕЕ СОСТОЯНИЕ: ANJOUAN (AOFA)

### 2.1 Что покрывает текущая лицензия

| Аспект | Детали |
|---|---|
| Регулятор | Anjouan Offshore Finance Authority (AOFA) |
| Тип лицензии | International Brokerage License |
| Разрешённая деятельность | Forex, CFD, товары, индексы, криптовалюты (как CFD) |
| Географический охват | Глобальный, за исключением ограниченных юрисдикций |
| Минимальный капитал | Согласно условиям лицензии [VERIFICAR CON ABOGADO] |
| Сегрегация средств | Требуется (раздельные счета) [VERIFICAR CON ABOGADO] |
| Compliance Officer | Требуется -- назначена Susana |
| KYC/AML | Требуется -- реализовано через Sumsub |
| Отчётность | Ежеквартально (AML) + ежегодно (финансовая) |
| Аудиты | Ежегодно (финансовый) |
| Продление | Ежегодно |

### 2.2 Преимущества Anjouan

- Относительно быстрый процесс получения.
- Низкая стоимость операционной деятельности по сравнению с юрисдикциями 1-го уровня.
- Регуляторная гибкость для инноваций и роста.
- Без ограничений кредитного плеча (в отличие от 1:30 в ЕС, 1:30 в Австралии для розничных клиентов).
- Без требований ICF (Investor Compensation Fund).

### 2.3 Ограничения Anjouan

- Восприятие более низкой достоверности среди искушённых трейдеров.
- Некоторые PSP и банки не работают с лицензиями Anjouan.
- Не позволяет прямой маркетинг в ЕС, Великобритании, Австралии (эти юрисдикции требуют местной лицензии).
- Не является членом ФАТФ (меньший вес в международном регулировании).
- У клиентов отсутствует компенсационный фонд.

## 3. MAURITIUS -- FINANCIAL SERVICES COMMISSION (FSC)

### 3.1 Общий обзор

Mauritius -- это уважаемая юрисдикция для офшорных финансовых услуг. FSC (Financial Services Commission) является финансовым регулятором небанковского сектора. Лицензия Mauritius значительно более признана, чем лицензия Anjouan, и открывает двери к лучшим банковским отношениям и PSP.

### 3.2 Необходимый тип лицензии

Для брокера forex/CFD необходима **Investment Dealer License (Full Service Dealer)** от FSC.

| Аспект | Детали |
|---|---|
| Регулятор | Financial Services Commission (FSC) Mauritius |
| Тип лицензии | Investment Dealer -- Full Service |
| Применимое законодательство | Securities Act 2005, Financial Services Act 2007 |
| Деятельность | Dealing in securities, forex, derivatives (включая CFD) |
| Географический охват | Глобальный |

### 3.3 Требования к капиталу

| Требование | Сумма | Примечания |
|---|---|---|
| Минимальный оплаченный капитал | USD 300,000 [VERIFICAR CON ABOGADO -- может варьироваться] | Должен быть размещён и подтверждён |
| Регуляторный чистый капитал | Поддерживать выше требуемого | Непрерывный мониторинг |
| Маржа платёжеспособности | Согласно расчёту FSC | Периодическая отчётность |

### 3.4 Требования Compliance

| Требование | Детали | Отличие от Anjouan |
|---|---|---|
| Compliance Officer | Выделенный, резидент Mauritius или с регулярным доступом | Может потребоваться местный сотрудник |
| MLRO (Money Laundering Reporting Officer) | Формально назначенный перед FSC | Более формализовано, чем в AOFA |
| Политика AML/KYC | Письменная, пересматриваемая ежегодно, утверждённая Board | Аналогично, но более строго |
| Risk assessment | Документированная оценка рисков бизнеса | Более детальная, чем в AOFA |
| Аудиты | Ежегодный (финансовый) + аудит compliance | Дополнительный аудит compliance |
| Отчётность | Ежеквартально + ежегодно + ситуативно | Аналогичная периодичность, больше деталей |
| Обучение | Обязательно для всего персонала | Аналогично тому, что NEOMAAA уже планирует |
| Сегрегация средств | Обязательные сегрегированные счета | Строже, под надзором |
| Тест пригодности клиента | Не строго обязателен для всех [VERIFICAR CON ABOGADO] | Зависит от типа клиента |

### 3.5 Структурные требования

| Требование | Детали |
|---|---|
| Местный офис | Требуется физический офис в Mauritius [VERIFICAR CON ABOGADO] |
| Местные директора | Минимум 2 директора -- резиденты Mauritius [VERIFICAR CON ABOGADO] |
| Управляющая компания | Может потребоваться местная management company |
| Местный аудитор | Аудиторская фирма, зарегистрированная в Mauritius |
| Зарегистрированный агент | Требуется |
| Страхование | Страхование профессиональной ответственности (PI insurance) |

### 3.6 Процесс получения

| Фаза | Деятельность | Ориентировочная длительность |
|---|---|---|
| 1. Предварительная заявка | Консультация с FSC, подготовка документации | 1-2 месяца |
| 2. Подача заявки | Формальная подача со всеми документами | 1 месяц |
| 3. Рассмотрение | FSC рассматривает заявку, может запросить дополнительную информацию | 2-4 месяца |
| 4. Условное одобрение | FSC выдаёт одобрение при выполнении условий | 1 месяц |
| 5. Выполнение условий | Капитал размещён, офис открыт, директора назначены | 1-3 месяца |
| 6. Выдача лицензии | Лицензия активна | -- |
| **Итого ориентировочно** | | **6-12 месяцев** |

## 4. SEYCHELLES -- FINANCIAL SERVICES AUTHORITY (FSA)

### 4.1 Общий обзор

Seychelles -- одна из наиболее популярных офшорных юрисдикций для брокеров forex. FSA (Financial Services Authority) предъявляет менее строгие требования, чем Mauritius, однако при более высокой репутации, чем Anjouan. Многие известные брокеры имеют структуру в Seychelles (IC Markets, Exness, Pepperstone, XM).

### 4.2 Необходимый тип лицензии

**Securities Dealer License (SD License)** от FSA.

| Аспект | Детали |
|---|---|
| Регулятор | Financial Services Authority (FSA) Seychelles |
| Тип лицензии | Securities Dealer License |
| Применимое законодательство | Securities Act 2007 |
| Деятельность | Dealing in securities, forex, derivatives |
| Географический охват | Глобальный (за исключением резидентов Seychelles для отдельных продуктов) |

### 4.3 Требования к капиталу

| Требование | Сумма | Примечания |
|---|---|---|
| Минимальный оплаченный капитал | USD 50,000 [VERIFICAR CON ABOGADO -- может измениться] | Значительно ниже, чем Mauritius |
| Регуляторный чистый капитал | Поддерживать согласно требованию | Мониторинг |

### 4.4 Требования Compliance

| Требование | Детали | Отличие от Anjouan |
|---|---|---|
| Compliance Officer | Назначенный | Аналогично AOFA |
| MLRO | Назначенный | Более формализовано |
| Политика AML | Письменная, основанная на AML Act Seychelles | Более специфичная, чем AOFA |
| KYC | Обязательный, аналогичен международным стандартам | Аналогично |
| Аудиты | Ежегодный (финансовый) | Аналогично |
| Отчётность | Ежегодно + ситуативно | Менее частая, чем Mauritius |
| Сегрегация средств | Требуется | Аналогично |
| Адрес компании | В Seychelles (может быть registered office) | Требуется местное присутствие |

### 4.5 Структурные требования

| Требование | Детали |
|---|---|
| Зарегистрированный офис | Да, в Seychelles |
| Директора | Минимум 1 директор. Не требуется строго быть местным резидентом [VERIFICAR CON ABOGADO] |
| Аудитор | Утверждённая аудиторская фирма |
| Зарегистрированный агент | Требуется (registered agent в Seychelles) |
| Страхование | Рекомендуется, не всегда обязательно [VERIFICAR CON ABOGADO] |

### 4.6 Процесс получения

| Фаза | Деятельность | Ориентировочная длительность |
|---|---|---|
| 1. Предварительная заявка | Подготовка документации, due diligence директоров | 1-2 месяца |
| 2. Подача заявки | Подача в FSA | 1 месяц |
| 3. Рассмотрение | FSA обрабатывает заявку | 2-3 месяца |
| 4. Одобрение | Лицензия выдана (может быть условной) | 1 месяц |
| **Итого ориентировочно** | | **3-6 месяцев** |

## 7. ЧТО СТРОИТЬ СЕЙЧАС ДЛЯ ОБЛЕГЧЕНИЯ РАСШИРЕНИЯ

### 7.1 Руководящий принцип

Каждое действие по compliance, которое NEOMAAA внедряет сегодня с лицензией Anjouan, должно проектироваться с расчётом на повторное использование в следующей юрисдикции. Это означает:

1. **Документировать всё так, как если бы более строгий регулятор собирался проверять.**
2. **Использовать международные стандарты, а не минимум AOFA.**
3. **Строить модульные системы**, которые можно адаптировать, а не переделывать.

### 7.2 Инфраструктура для повторного использования

| Что строится сейчас | Пригодно для Seychelles | Пригодно для Mauritius | Пригодно для CySEC/FCA |
|---|---|---|---|

| Документированный процесс EDD | Да | Да | Да |
| Процесс SAR | Да (адаптировать к местному FIU) | Да (адаптировать к местному FIU) | Да (адаптировать к NCA/FIU) |
| Обучение команды | Да | Да | Да (более частое, с оценками) |
| Ежеквартальная отчётность | Да (адаптировать формат) | Да (адаптировать формат) | Да (более детальный формат) |

### 7.3 Конкретные действия для расширения

**Для Seychelles (меньше усилий):**
1. Поддерживать успешную работу текущей системы compliance в течение 6-12 месяцев (послужной список).

**Для Mauritius (больше усилий):**
