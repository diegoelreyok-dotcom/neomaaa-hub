## 4. Оптимизация лендинг-страниц

### 4.1 Элементы, конвертирующие в LATAM

На основе лучших практик сектора офшорных брокеров для аудитории LATAM:

**Above the fold (первые 5 секунд):**
- Чёткий заголовок с основным преимуществом (не фичами)
- Минимальный депозит виден ($5)
- Заметный CTA: "Открой счёт бесплатно"
- Сигналы доверия: регулирование, количество инструментов, локальные методы оплаты
- Локальный язык (нейтральный испанский, или португальский для Бразилии)

**Below the fold:**
- Логотипы методов оплаты конкретной страны (PSE для Колумбии, PIX для Бразилии и т.д.)
- Упрощённая сравнительная таблица счетов
- Отзывы или статистика использования
- Короткое видео онбординга (60-90 секунд)
- FAQ с типичными возражениями
- Видимый, но не навязчивый дисклеймер о рисках

### 4.3 Чеклист оптимизации (CRO)

- [ ] Время загрузки <3 секунд (особенно на мобильном — LATAM имеет переменное соединение)
- [ ] 100% адаптивный (70%+ трафика LATAM — мобильный)
- [ ] Форма регистрации максимум 4 начальных поля (имя, email, телефон, страна)
- [ ] Автоопределение страны для показа релевантных методов оплаты
- [ ] Видимый, но не навязчивый чат Intercom
- [ ] Дисклеймер о рисках в футере (обязателен)
- [ ] Установлены пиксели Meta, тег Google Ads, пиксель TikTok
- [ ] Отслеживаемые события конверсии: PageView, InitiateRegistration, CompleteRegistration, KYC, FTD
- [ ] Активные тепловые карты (Hotjar или Microsoft Clarity — оба бесплатны)
- [ ] Плавающий WhatsApp для рынков, где это доминирующий канал (Колумбия, Бразилия, Мексика)

| Тактика | Ожидаемый эффект | Усилие | Приоритет |

| Copy Trading как крючок: "Пополните $50 и скопируйте нашего топ-трейдера" | +10-15% | Среднее | P2 |

**Критическая сегментация для KYC → FTD:**
- По стране: адаптировать сообщение и выделяемый метод оплаты
- По происхождению: кроссовер проп vs платная реклама vs органический (разные сообщения)
- По поведению после KYC: вернулся на сайт vs не вернулся

### 6.2 Методы оплаты LATAM как конкурентное преимущество

Многие конкуренты (IC Markets, Pepperstone) не предлагают локальные методы LATAM. Эксплуатировать это:

| Страна | Выделяемые методы | Сообщение |

### 7.2 Правила A/B тестирования

- Минимум 500 пользователей на вариант до объявления победителя
- Минимальная статистическая значимость: 95%
- Запускать тест минимум 7 дней (захватить недельный паттерн)
- Тестировать только 1 переменную за раз
- Документировать все тесты и результаты в общем журнале
- Тесты, затрагивающие рекламу, должны длиться 14+ дней для стабильности алгоритмов

**Строка 1 — Объём:**
- Посетители сегодня / неделя / месяц

- FTD сегодня / неделя / месяц

**Строка 2 — Конверсия:**
- Трафик → Регистрация (%)
- Регистрация → KYC (%)
- KYC → FTD (%)
- FTD → Активный (%)

**Строка 3 — Эффективность:**
- CPA по каналам
- ROAS по каналам
- Средний депозит
- Проектируемый LTV

**Строка 4 — Пайплайн:**
- Регистрации с незавершённым KYC (очередь)
- Одобренные KYC без FTD (очередь)
- FTD без первой операции (очередь)
- Клиенты в зоне риска оттока
